# SynOS Landing Page

## Deployment
- Vercel auto-deploys from `main` branch
- Vercel serves from `public/` directory (configured in `vercel.json`)
- **Always update both `index.html` AND `public/index.html`** when making changes
- Live at: https://www.synos.in

## Structure
- Single-page site, pure HTML/CSS/JS, no build step
- All code in `index.html` (~1500 lines)
- Form submissions go to Web3Forms

## Git
- Repo: github.com/synoshq/synos.in
- Branch: `main`
